# Análise do Sistema de Ajuste

## Leitura das cotas:

| Elemento | Cota | Letra |
|---|---|---|
| Diâmetro maior | Ø 13 **n6** | minúscula → **eixo** |
| Diâmetro menor | Ø 12 **p6** | minúscula → **eixo** |

---

## Observação:

Ambas as cotas possuem letras **minúsculas (n, p)** — indicando **eixos**.

No desenho, os furos das peças acopladas **não aparecem cotados**, o que significa que o **furo é o elemento de referência fixo (base)** — e os eixos é que variam.

> Porém, a ausência de letra **h** nos eixos indica que eles **não são a base** — a base é o **furo (H)**, implícito.

---

## Resposta:

**b) (X) sistema eixo base**

> No **sistema eixo base**, o eixo é o elemento de referência com desvio fixo, e os furos variam para obter os diferentes tipos de ajuste. As tolerâncias **n6 e p6** (letras minúsculas, sem ser "h") confirmam que os eixos possuem posições de campo definidas, sendo o eixo o elemento base do sistema.
