# Análise do Ajuste

## Leitura das cotas:

| Peça | Cota | Tipo |
|------|------|------|
| **Furo** (hachurado) | Ø 28 +0,021 / 0 | Furo |
| **Eixo** (liso) | Ø 28 +0,035 / +0,022 | Eixo |

---

## Calculando os limites:

### Furo:
- Máximo = 28 + 0,021 = **28,021 mm**
- Mínimo = 28 + 0 = **28,000 mm**

### Eixo:
- Máximo = 28 + 0,035 = **28,035 mm**
- Mínimo = 28 + 0,022 = **28,022 mm**

---

## Comparando:

```
Eixo mínimo  (28,022)  >  Furo máximo (28,021)
```

> O eixo é **sempre maior** que o furo em todas as situações possíveis!

---

## Conclusão:

**a) (X) ajuste com interferência**

O eixo sempre será maior que o furo, exigindo **força ou prensagem** para montagem — característica do ajuste com interferência.
